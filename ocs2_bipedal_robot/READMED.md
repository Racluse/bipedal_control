# ocs2_bipedal_robot

This package contains the OCS2 interface for the bipedal robot.

It is based on the [ocs2_legged_robot](https://github.com/leggedrobotics/ocs2/tree/main/ocs2_robotic_examples/ocs2_legged_robot) package.
